## Final Project 

Welcome to my Finall project Bank Account 

## Author: Eric Orea

The workspace contains two folders by default, where:

- `src`: the folder to maintain sources
- `doc`: the folder to maintain documentation related to my project (UML diagram, etc)

This project is a Java-based banking application consisting of two primary classes:

BankAccount: Manages individual account data, including balance, deposits, and 
withdrawals.

BankApp: The entry point of the application that provides a user interface to interact with the bank accounts.

BankAccountGUI: Makes a graphical user interface that you can interact with and will be able to withdraw ect.

Features:
Creates new bank accounts.
Deposits and withdraws funds with balance validation.
Views current account statements.
ect.
